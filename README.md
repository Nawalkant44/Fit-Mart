## **How to run app locally**

```
git clone https://github.com/aman11s/fitMart.git
cd fitMart
npm install
npm start
```



## **Key Features**

- Homepage
- Product listing page
- Filters by
  - Price
  - Category
  - Ratings
  - Price - High to Low
  - Price - Low to High
- Wishlist Management
- Cart Management
- Authentication
  - Login
  - Logout
  - Signup
- Single Product Page
- Loading and Alerts
- User Profile Page
